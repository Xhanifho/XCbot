Ikuti langkah mudah ini untuk dapatkan akses bot:

Buka Website: Pergi ke laman web tempat kau main Wingo dan Login akaun macam biasa.

Buka Developer Tools: Tekan butang F12 pada keyboard (atau Right Click > Inspect).

Masuk Menu Network: Pilih tab Network di bahagian atas panel yang baru muncul.

Filter Socket: Pada kotak carian (filter), taip perkataan "wingo" atau terus klik pada sub-tab WS (Websocket).

Cari Fail WS: Cari nama fail yang muncul (biasanya bermula dengan ws atau alamat websocket), kemudian klik padanya.

Salin Cookies:

Klik pada tab Headers.

Skrol ke bawah cari bahagian Request Headers.

Cari baris bertulis cookie:.

Copy (Salin) semua kod panjang di sebelah tulisan cookie tersebut dan paste ke dalam bot.
